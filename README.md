# Сканер штрих-кодов через веб-камеру

Простое веб-приложение на чистом HTML/JS, которое использует:
- `getUserMedia` для доступа к камере;
- `BarcodeDetector API` для распознавания штрих-кодов в реальном времени.

## Как запустить

### 1) Открой терминал в папке проекта
Если ты уже находишься в проекте, просто проверь путь:

```bash
pwd
```

Если нужно перейти в папку проекта вручную:

```bash
cd /workspace/barcode_test
```

### 2) Убедись, что Python установлен

```bash
python3 --version
```

Если команда выводит версию (например, `Python 3.10.x`) — всё готово.

### 3) Запусти локальный HTTP-сервер

```bash
python3 -m http.server 8000
```

Что это значит:
- `python3 -m http.server` — встроенный веб-сервер Python;
- `8000` — порт, на котором будет доступен сайт.

После запуска в терминале появится сообщение вроде:
`Serving HTTP on 0.0.0.0 port 8000 ...`

### 4) Открой приложение в браузере
Перейди по адресу:

- `http://localhost:8000`

### 5) Начни сканирование
1. Нажми **Старт**.
2. Разреши доступ к камере.
3. Поднеси штрих-код к центру экрана.

### 6) Как остановить сервер
Вернись в терминал, где запущен сервер, и нажми:

- `Ctrl + C`

## Поддерживаемые форматы

`ean_13`, `ean_8`, `upc_a`, `upc_e`, `code_128`, `code_39`, `codabar`, `itf`, `qr_code`.

> Лучше всего работает в актуальном Chrome/Edge, где есть поддержка `BarcodeDetector`.
