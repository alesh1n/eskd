# Классификатор ЕСКД

Простой статический сайт на `HTML`, `CSS` и чистом `JavaScript`, который работает как диалоговый помощник для подбора децимального номера по дереву вопросов.

Интерфейс устроен как чат:
- система показывает вопрос;
- пользователь отвечает кнопками `Да` и `Нет`;
- после прохождения дерева решений показывается итоговый результат.

## Файлы

- `index.html` - разметка интерфейса.
- `style.css` - стили и анимации.
- `app.js` - логика чата, загрузка дерева решений и переходы по ответам.
- `classifier.json` - дерево вопросов и результатов.

## Как запустить

Важно: сайт нужно открывать через локальный HTTP-сервер. Если открыть `index.html` напрямую через `file://`, браузер может заблокировать загрузку `classifier.json`.

### Вариант через Python

Открой терминал в папке проекта:

```powershell
cd C:\Users\aaleshin\yeskd
python -m http.server 8000
```

После этого открой в браузере:

```text
http://localhost:8000
```

## Как это работает

`app.js` загружает `classifier.json` и начинает диалог с корневого узла.

Каждый узел дерева может быть двух типов:

1. Узел с вопросом
2. Конечный узел с результатом

Пример структуры:

```json
{
  "id": "root",
  "question": "Это деталь?",
  "yes": {
    "question": "Это корпусная деталь?",
    "yes": {
      "result": "123456"
    },
    "no": {
      "result": "345678"
    }
  },
  "no": {
    "result": "456789"
  }
}
```

## Как добавлять новые вопросы

Чтобы расширить классификатор, достаточно редактировать `classifier.json`.

Правила:
- у узла с вопросом должно быть поле `question`;
- у него должны быть ветки `yes` и `no`;
- конечный узел содержит поле `result`;
- значение `result` и есть итоговый децимальный номер.

Пример:

```json
{
  "question": "Это сборочная единица?",
  "yes": {
    "result": "456789"
  },
  "no": {
    "question": "Это стандартное изделие?",
    "yes": {
      "result": "567890"
    },
    "no": {
      "result": "678901"
    }
  }
}
```

## Особенности

- Без фреймворков и библиотек.
- Вся логика ветвления хранится отдельно в `classifier.json`.
- Интерфейс легко адаптировать под другой классификатор.
- Вопросы и ответы отображаются в виде диалога.
